# AstroEngine

Rails engine that provides simple ASTRO data.

COMING SOON

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'astro_engine'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install astro_engine
```
